**Dopisy ze Severu**

V tomto projektu najdete mé texty psané na Islandu mezi roky 2017–2025.


**Živá stránka**  

👉 [magdiroubalova-eng.github.io/dopisy_ze_severu]


**O projektu** 

Dopisy ze Severu => osobní literární projekt: sbírka textů psaných na Islandu během mých více či méně pracovních;
více či méně dlouhodobých návštěv.
Vznikl jako HTML/CSS webová stránka s důrazem na čistou typografii a atmosféru severské krajiny.

**Struktura projektu** 


```
dopisy_ze_severu/
├── index.html        # Hlavní stránka
├── css/
│   └── style.css     # Styly
├── img/              # Fotografie
├── posts/            # Texty (ignorovány Gitem)
└── README.md
```

**Vlastnosti**

- Minimalistický design inspirovaný islandskou krajinou
- Černobílé fotografie
- Hamburger menu
- Responzivní layout
